Как запустить
1. Установите зависимости для бэкенда:
bash
pip install fastapi uvicorn sqlalchemy
2. Запустите бэкенд:
bash
cd backend
python main.py 

Особенности реализации
Backend:

FastAPI для быстрого и удобного создания API

SQLAlchemy ORM для работы с базой данных

SQLite для простоты развертывания

Автоматическая документация OpenAPI

Поддержка CORS для фронтенда

Frontend:

Чистый JavaScript без фреймворков

Модульная архитектура с классом TaskManager

Адаптивный дизайн с CSS Grid/Flexbox

Интуитивно понятный интерфейс

Обработка ошибок и уведомления

Подтверждение удаления задач

Функциональность:

Создание, чтение, обновление, удаление задач

Фильтрация задач (все/активные/выполненные)

Статистика по задачам

Валидация форм

Анимации и плавные переходы
